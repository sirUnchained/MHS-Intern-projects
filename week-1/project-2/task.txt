پروژه ۲: ایجنت تحلیل طلا (Gold Analysis Agent)

هدف پروژه

طراحی یک ایجنت هوشمند برای پاسخگویی به سوالات کاربران درباره بازار طلا با استفاده از داده‌های به‌روز بازار، اخبار مالی و مدل‌های زبانی (LLM).

قابلیت‌های مورد نیاز

1. دریافت پیام کاربر

ایجنت باید بتواند سوالات کاربر را دریافت کند.

نمونه سوالات:

- قیمت فعلی طلا چقدر است؟
- روند طلا در هفته آینده چگونه است؟
- آیا الان زمان مناسبی برای خرید طلا است؟
- اخبار مهم امروز درباره طلا چیست؟

2. دریافت اطلاعات بازار

ایجنت باید اطلاعات بازار را از یک یا چند منبع دریافت کند.

منابع پیشنهادی:

- Tavily Search API
- Yahoo Finance
- Finnhub API
- Alpha Vantage (اختیاری)

3. تحلیل اخبار

ایجنت باید بتواند اخبار مرتبط با طلا را دریافت و خلاصه‌سازی کند.

موارد مورد بررسی:

- اخبار اقتصادی
- نرخ بهره آمریکا
- شاخص دلار (DXY)
- تنش‌های ژئوپلیتیکی
- گزارش‌های تورمی

4. استفاده از مدل زبانی (LLM)

برای تولید پاسخ نهایی از یکی از مدل‌های زیر استفاده شود:

- OpenAI GPT
- Gemini
- مدل‌های متن‌باز

5. تولید پاسخ نهایی

ایجنت باید با ترکیب:

- داده‌های بازار
- اخبار
- تحلیل مدل زبانی

یک پاسخ قابل فهم برای کاربر تولید کند.

نمونه خروجی:

"قیمت فعلی اونس طلا 3,350 دلار است. با توجه به اخبار اخیر مربوط به کاهش نرخ بهره و افزایش تقاضای سرمایه‌گذاران، روند کوتاه‌مدت طلا صعودی ارزیابی می‌شود. با این حال این پاسخ توصیه مالی محسوب نمی‌شود."


کتابخانه‌های پیشنهادی

- requests
- pandas
- openai / google-generativeai
- yfinance
- tavily-python

فایل‌های تحویلی

1. فایل Jupyter Notebook
2. کد کامل پروژه
3. فایل ارائه (Presentation)
4. نمونه خروجی چند پرسش کاربر

نکات مهم

- کدها باید تمیز و کامنت‌گذاری شده باشند.
- مدیریت خطا برای APIها انجام شود.
- پاسخ‌ها باید بر اساس داده واقعی تولید شوند.
- استفاده از Prompt Engineering امتیاز مثبت دارد.

امتیاز اضافه

- رابط کاربری با Streamlit
- ذخیره تاریخچه گفتگو
- نمایش نمودار قیمت طلا
- پشتیبانی از زبان فارسی و انگلیسی
